# Översikt över användningsfall i Personalization

Med Adobe Experience Platform Web SDK kan företag leverera en mängd olika typer av personaliserade användningsfall till sina webbegenskaper. Ni kan smidigt koppla samman datainsamling, beslut och aktiveringsprocesser i olika Adobe-lösningar som Target, Journey Optimizer och Audience Manager.

SDK har stöd för flexibla arkitekturer, inklusive klient-, server- och hybridmodeller, vilket gör det möjligt att hämta och återge personaliserade erbjudanden som uppfyller de flesta av organisationens användningsfall.

## SDK-exempel

Förutom dokumentsidorna i den här mappen har Adobe en databas med exempelprogram som du kan referera till. Se [SDK-exempel på webben](https://github.com/adobe/alloy-samples/) på GitHub om du vill ha fler användningsexempel för personalisering, bland annat:

* Personalisering på klientsidan
* Personalisering på serversidan
* Hybrid-personalisering
* Personalization med ensidiga program
