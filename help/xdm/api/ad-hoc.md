# Ad hoc-scheman

Under särskilda omständigheter kan det vara nödvändigt att skapa ett [!DNL Experience Data Model] (XDM)-schema med fält som namnges endast för användning av en enda datauppsättning. Detta kallas ett ad hoc-schema. Ad-hoc-scheman används i flera arbetsflöden för dataöverföring, inklusive inhämtning av CSV-filer och skapande av vissa typer av källkopplingar. Mer information om ad hoc-scheman, inklusive anvisningar om hur du skapar ett i [!DNL Schema Registry] API:t, finns i [ad hoc-schematusjälvstudiekursen](../tutorials/ad-hoc.md).
