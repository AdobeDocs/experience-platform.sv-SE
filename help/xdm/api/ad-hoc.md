# Ad hoc-scheman

Under särskilda omständigheter kan det vara nödvändigt att skapa ett XDM-schema (Experience Data Model) med fält som bara namnges av en enda datauppsättning. Detta kallas för ett ad hoc-schema. Ad-hoc-scheman används i flera arbetsflöden för dataöverföring, inklusive inhämtning av CSV-filer och skapande av vissa typer av källkopplingar. Mer information om ad hoc-scheman, inklusive anvisningar om hur du skapar ett, finns i [ad hoc-schemautstudiekursen](../tutorials/ad-hoc.md).
