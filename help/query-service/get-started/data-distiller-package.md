# Distiller-paket

Data Distiller är ett paket som innehåller en deluppsättning av funktionerna från Adobe Experience Platform. Det här paketet är avsett för dataförberedelse efter konsumtion i realtid för kundprofiler eller för användning av analysdata genom batchfrågor som körs i Query Service. Mer information om Data Distiller-paketet finns i [Distiller-översikten](../data-distiller/overview.md).
