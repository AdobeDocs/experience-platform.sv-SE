# ETL-arbetsflöde

Olika extraherings-, omformnings- och inläsningsverktyg (ETL) kan ha olika format, men de flesta exponerar liknande funktioner för att underlätta kombinationen av flera datakällor.

## Exempel på ETL-verktyg
