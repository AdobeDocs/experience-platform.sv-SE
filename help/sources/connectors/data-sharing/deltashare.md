# [!DNL Deltashare]

Du kan använda [!DNL Deltashare]-källanslutningen för att upprätta en säker anslutning till [!DNL Databricks Delta Sharing]-servrar och skapa virtuella datauppsättningar i Adobe Experience Platform. Med den här funktionen kan du söka efter virtuella datauppsättningar, inspektera och mappa externa tabeller till XDM-scheman (Experience Data Model) och utnyttja dem i olika Experience Platform-program och -tjänster. Genom [!DNL Deltashare] kan du söka efter fysiskt inmatning av datauppsättningar och därigenom minska dubbletter, lagringskostnader och driftskostnader.
