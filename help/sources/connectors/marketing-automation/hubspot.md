# (Beta) HubSpot-anslutning

>[!NOTE]
>HubSpot-anslutningen är i betaversion. Mer information om hur du använder betatecknade anslutningar finns i [Källor-översikten](../../home.md#terms-and-conditions) .

Adobe Experience Platform tillåter att data hämtas från externa källor samtidigt som du kan strukturera, märka och förbättra inkommande data med hjälp av Platform tjänster. Du kan importera data från en mängd olika källor, till exempel Adobe-program, molnbaserad lagring, databaser och många andra.

Experience Platform stöder inmatning av data från ett system för automatisering av tredjepartsmarknadsföring. Stöd för leverantörer av automatiserad marknadsföring innefattar HubSpot.
