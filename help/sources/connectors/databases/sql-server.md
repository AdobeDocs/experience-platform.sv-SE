# (Beta) Microsoft SQL Server-anslutning

Med Adobe Experience Platform kan data hämtas från externa källor samtidigt som ni kan strukturera, märka och förbättra inkommande data med hjälp av plattformstjänster. Du kan importera data från en mängd olika källor, till exempel Adobe-program, molnbaserad lagring, databaser och många andra.

Experience Platform har stöd för inmatning av data från en tredjepartsdatabas. Plattformen kan ansluta till olika typer av databaser, till exempel relationsdatabaser, NoSQL-databaser eller datalager. Stöd för databasproviders är bland annat Microsoft SQL Server.

Dokumentationen nedan innehåller information om hur du ansluter Microsoft SQL Server till plattformen med API:er eller användargränssnittet:
