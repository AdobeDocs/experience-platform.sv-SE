# (Beta) Couchbase-kontakt

>[!NOTE]
>Couchbase-kopplingen är i betaversion. Mer information om hur du använder betatecknade anslutningar finns i [Källor-översikten](../../home.md#terms-and-conditions) .

Adobe Experience Platform erbjuder inbyggd anslutning för databasleverantörer som Microsoft, MySQL och Azure, vilket gör att du kan hämta data från dessa system. Olika typer av tredjepartsdatabaser stöds, bland annat relational, NoSQL och data warehouse. Stöd för databasleverantörer inkluderar Couchbase.

Dokumentationen nedan innehåller information om hur du ansluter Couchbase till Platform med hjälp av API:er eller användargränssnittet:

- [Samla in data från en databas med API:t för Flow Service](../../tutorials/api/collect/database-nosql.md)
