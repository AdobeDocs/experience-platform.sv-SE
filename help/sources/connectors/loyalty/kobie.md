# [!DNL Kobie]

[!DNL Kobie] innehåller en sammansättningsbar SaaS-plattform som du kan använda för att hantera hela livscykeln för ett lojalitetsprogram. Plattformen [!DNL Kobie] används oftast av marknadsföringsteam, datavetare och kundupplevelseansvariga och innehåller ett kodfritt gränssnitt som du kan använda för att samordna komplexa lojalitetsregler.

Du kan använda [!DNL Kobie]-källorna i Adobe Experience Platform-källkatalogen för att importera både batch- och direktuppspelade lojalitetsdata från ditt [!DNL Kobie]-konto.

- [[!DNL Kobie] batch](../../tutorials/ui/create/loyalty/kobie-batch.md)
- [[!DNL Kobie] direktuppspelning](../../tutorials/ui/create/loyalty/kobie-streaming.md)
