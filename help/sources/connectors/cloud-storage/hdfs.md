# (Beta) Apache HDFS-anslutning

>[!NOTE]
>Apache HDFS-kontakten är i betaversion. Mer information om hur du använder betatecknade anslutningar finns i [Källor-översikten](../../home.md#terms-and-conditions) .

Adobe Experience Platform erbjuder inbyggd anslutningsbarhet för molnleverantörer som AWS, Google Cloud Platform och Azure, vilket gör att du kan hämta data från dessa system. Inkapslade data kan formateras som JSON, parquet eller avgränsade. Stöd för molnlagringsleverantörer inkluderar Apache HDFS.

Dokumentationen nedan innehåller information om hur du ansluter Apache HDFS till Platform med API:er eller användargränssnittet:

- [Samla in molnlagringsdata med API:t för Flow Service](../../tutorials/api/collect/cloud-storage.md)
