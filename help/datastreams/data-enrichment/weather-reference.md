# Mappningar av väderdatafält

Adobe har samarbetat med [!DNL [The Weather Company]](https://www.ibm.com/weather) för att ge den ytterligare kontexten för USA:s väder till data som samlats in via datastreams. Ni kan använda dessa data för analyser, målinriktning och segmentskapande i Experience Platform.

På den här sidan visas alla tillgängliga fält som du kan använda för att förbättra dina målgruppsdata.

Fälten delas upp i tre olika grupper som justeras efter fältgrupperna.

* [**[!UICONTROL Current Weather]**](#current-weather): Användarens aktuella väderförhållanden, baserat på deras plats. Detta inkluderar aktuell temperatur, kryptering, molntäckning med mera.
* [**[!UICONTROL Forecasted Weather]**](#forecast): Prognosen innehåller prognosen 1,2,3,5,7 och 10 dagar för användarplatsen.
* [**[!UICONTROL Triggers]**](#triggers): Utlösare är specifika kombinationer som mappar till olika semantiska väderförhållanden. Det finns tre olika typer av väderutlösare:

   * **[!UICONTROL Relative Triggers]**: Semantiskt meningsfulla förhållanden, till exempel kallt eller regnigt väder. Dessa kan skilja sig åt i definitioner mellan olika klimat.
   * **[!UICONTROL Product Triggers]**: Villkor som skulle leda till inköp av olika typer av produkter. Exempel: väderprognoser kan innebära att inköp av regnrockar är mer sannolika.
   * **[!UICONTROL Severe Weather Triggers]**: Allvarliga vädervarningar, som vinterstorm eller orkanvarningar.

| Fält | Beskrivning | XDM-sökväg |
| --- | ---- | --- |
| [!UICONTROL Product Triggers] | Anger när villkoren är lämpliga för att öka försäljningen för vissa produktkategorier.  De mappas till heltalsobjekt. En fullständig lista finns på IBM. | `weather.productTriggers` |
| [!UICONTROL Relative Triggers] | Relativa förhållanden baserade på människans uppfattning. Som varmt eller kallt. De mappas till heltalsobjekt. En fullständig lista finns på IBM. | `weather.relativeTriggers` |
| [!UICONTROL Severe Weather Triggers] | Visar olika allvarliga väderförhållanden som Hurricanes eller överdrivet regn. | `weather.severeTriggers` |
